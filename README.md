# Project Guide

## clone repo from github

```bash
git clone https://github.com/dvlprroshan/digitalcard/
```

## change current path
```bash
cd digitalcard
```

## for installing dependencies in project

```bash
npm i
```

## for running project in development server

```bash
npm run dev
```

## building project

```bash
npm run build
```

## running production build

```bash
npm start
```
